Программирование

Выберите язык, на котором хотите посмотреть примеры программ

Pascal

Паскаль является учебным языком программирования. Его чаще всего начинают изучать в школе и он прекрасно подходит для понимании основ программирования

C#

C# - язык программирования для создании десктопных приложений. Также с использованием ASP.NET можно создать серверную часть сайта

JavaScript

JavaScript - язык, предназначенный для web-разработки. Также этот язык считается самым востребованным. Ведь на нём помимо придания сайтам динамичноти, есть возможность создания приложений как на клиентской стороне, так и на серверной

PHP

PHP является серверным языком программирования. Поэтому применяется обычно с системами управления базами данных, к примеру для обеспечения регистрации пользователя
